# Buy & Sell

An E-commerce application to buy and sell products built using Django

# Tools and Technologies Used 

- Django

- Python

- SQLite

- HTML

- TailwindCSS

- Jinja Templating Language

- Javascript

- Docker (for containerization)

- Stripe (Payment gateway)
